# Evolution simulator


This is a simple evolution by natural selection simulator, made in unity C#


### Important things

population_dashboard.html is a file (made by claude) that reads the csv file that the simulation gives and it plots some graphs based on time.
The output csv can be found at ""C:\Users\YOURNAME\AppData\LocalLow\DefaultCompany\Evolution" on windows, on linux I have no idea.

**DO DELETE THE FILE EACH SIMULATION, AS THE CSV IS WRITTEN ADITIVIELY, WHICH CAUSES SUDDEN SPIKES AND DROPS ON THE GRAPHS**


**AT THE MOMENT THERE IS NO WAY TO TUNE THE SIMULATION, IT RUNS WITH 100 INDIVIDUOUS AT THE START AND 60 RESOURCES SCATTERED AROUND THE MAP, IT MAY LAG**
